# SGP4Position (IEntityPosition2)

SGP4/TLE 卫星位置对象，用于根据两行轨道根数（TLE）计算卫星轨道。

## 继承关系
- 实现：`IEntityPosition2`
- `$type`: `SGP4`

## 字段说明

| 字段名 | 类型 | 必须 | 默认值 | 说明 |
| :--- | :--- | :--- | :--- | :--- |
| `$type` | string | 是 | `SGP4` | 类型标识符 |
| `SatelliteNumber` | string | 否 | `25730` | 卫星 SSC number |
| `TLEs` | string[] | 是 | - | 卫星星历 (TLE)，2 行字符串数组 |

## TLE 格式

TLE (Two-Line Element) 是 NORAD 标准的轨道根数格式：

```
1 25730U 99025A   21120.62396556  .00000659  00000-0  35583-3 0  9997
2 25730  99.0559 142.6068 0014039 175.9692 333.4962 14.16181681132327
```

## JSON 示例

```json
{
  "$type": "SGP4",
  "SatelliteNumber": "25730",
  "TLEs": [
    "1 25730U 99025A   21120.62396556  .00000659  00000-0  35583-3 0  9997",
    "2 25730  99.0559 142.6068 0014039 175.9692 333.4962 14.16181681132327"
  ]
}
```

## 注意事项

- TLE 数据必须为有效的 NORAD 格式
- 适用于近地轨道卫星
- 考虑大气阻力、地球扁率等摄动
