# EntityPositionSGP4 数据结构

SGP4/TLE 卫星位置对象，含 `$type` 判别字段（值为 `SGP4`）。

## 字段说明


| 字段名               | 类型       | 必须   | 默认值            | 说明                            |
| ----------------- | -------- | ---- | -------------- | ----------------------------- |
| `$type`           | string   | 是    | `SGP4`         | 类型标识符，必须为 `SGP4`              |
| `SatelliteNumber` | string   | null | 否              | `25730`                       |
| `TLEs`            | string[] | 是    | （见 OpenAPI 示例） | 两行 TLE 字符串数组 `[line1, line2]` |


## JSON 示例

```json
{
  "$type": "SGP4",
  "SatelliteNumber": "25730",
  "TLEs": [
    "1 25730U 99025A   21120.62396556  .00000659  00000-0  35583-3 0  9997",
    "2 25730  99.0559 142.6068 0014039 175.9692 333.4962 14.16181681132327"
  ]
}
```

## 注意事项

- `$type` 必须为 `SGP4`，否则无法作为该多态分支正确反序列化。
