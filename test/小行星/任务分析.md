# 小行星转移窗口任务分析

## 任务目标

针对 `2024 SJ3`、`2025 WH5`、`2025 WN6`、`2024 QM` 四颗小行星,使用仓库内 `celestial-ephemeris`、`celestial-mpc`、`celestial-transfer` 技能对应的 Astrox WebAPI 完成以下计算:

1. 计算 2029-01-01 至 2029-12-31 小行星与地球的日心星历。
2. 统计小行星与地球距离随时间的变化。
3. 选取距离小于等于 1800 万 km 的连续日期段作为候选到达窗口。
4. 在地球出发时间 2028-03-01 至 2028-12-31 内,对候选到达窗口进行 Lambert 转移搜索。
5. 选取地球出发端速度增量 `DV1_Mag` 最小的转移结果。
6. 使用最优结果中的到达位置矢量 `RV2[0:3]` 与到达相对速度矢量 `DeltaV2` 计算夹角,作为到达光照角。

## API 与数据保存约定

- 行星星历: `POST /celestial/ephemeris`,用于获取地球 2029 全年日心 `MEANECLPJ2000` 星历。
- 小行星星历与轨道根数: `POST /celestial/mpc`,用于获取小行星 MPC 根数及递推星历。
- 转移搜索: `POST /celestial/transfer`,使用小行星 `ArrivalElements` 避免转移阶段重复 MPC 查询。
- 所有请求保存为 `*-input.json`,响应保存为 `*-output.json`。
- 每颗小行星生成 `任务分析结果-[小行星名称].md`。

## 计算假设

- 星历步长使用 1 天,距离筛选基于每日采样点。
- 转移搜索步长默认使用 5 天,用于在全年窗口内控制 API 计算量。
- 若某颗小行星在 2029 年没有进入 1800 万 km 阈值,则不进行转移搜索。
- 到达光照角定义为最优转移结果中 `RV2` 位置矢量与 `DeltaV2` 到达相对速度矢量的夹角,范围为 0-180 deg。

## 可复现执行

运行:

```bash
python test/小行星/asteroid_transfer_analysis.py
```

输出文件均位于 `test/小行星` 目录下。
