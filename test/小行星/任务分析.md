# 小行星任务分析说明

任务范围与步骤见同目录 `任务-0427.md`（星历、距离窗口、Lambert 转移、到达光照角等）。

## 地球出发等效 dV 算法（约定）

以下用于把 Astrox 转移结果中的 **`DV1_Mag`（m/s）** 换算为「相对地球、200 km 近地点高度」语境下的 **地球出发 dV**，单位 **km/s**。本算法为分析约定，便于与近地圆轨道参考速度对比。

1. **将 `DV1_Mag` 视为相对地球的无穷远超速大小**  
   \[
   v_\infty\,[\mathrm{km/s}] = \frac{\mathrm{DV1\_Mag}\,[\mathrm{m/s}]}{1000}
   \]

2. **由双曲线能量求 200 km 高度近地点速度大小**（地心二体，地球 \(\mu\)，赤道半径 \(R_E\)，近地点高度 \(h = 200\,\mathrm{km}\)）  
   \[
   r_p = R_E + h,\quad
   v_p = \sqrt{\,v_\infty^2 + \frac{2\mu}{r_p}\,}
   \]
   常数取：\(\mu = 398600.4418\,\mathrm{km^3/s^2}\)，\(R_E = 6378.137\,\mathrm{km}\)，故 \(r_p = 6578.137\,\mathrm{km}\)。

3. **地球出发 dV**  
   \[
   \Delta v_{\mathrm{earth}} = v_p - 10.96\,\mathrm{km/s}
   \]
   其中 **10.96 km/s** 为约定的参考量（与 200 km 高度圆轨道速度同量级，用于扣减后得到「相对该参考」的出发 dV）。

**说明**：未计入大气、地球非球形及第三体；若物理上 `DV1_Mag` 与日心 Lambert 脉冲不完全等同地球系 \(v_\infty\)，仍以本约定作为任务内统一口径。
