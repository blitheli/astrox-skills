## 4. 到达光照角

按任务书：取到达时刻 **`RV2` 前三个分量** 为位置矢量 **r**，**`DeltaV2`** 为到达脉冲矢量，光照角 **θ** 为 **r** 与 **Δv₂** 的夹角（单位：度），按下式计算：

`cos θ = (r·Δv₂) / (|r| |Δv₂|)`，`θ = arccos(cos θ)`（结果裁剪到 [-1, 1] 避免数值误差）。

对全部 **7344** 条 `TransferResults` 在后处理中计算 **θ**。每条记录的 `IlluminationAngleDeg` 见脚本导出（撞击器候选、配对 JSON 等）。

## 5. 到达时刻地球–小行星距离

用于满足补充约束：**撞击器到达小行星时，地球与该小行星的距离尽量小**。

- **地球位置**：`POST /celestial/ephemeris`，`TargetName=Earth`，`ObserverName=Sun`，`ObserverFrame=MEANECLPJ2000`，时间窗覆盖全部可能到达时刻（本分析为 `2028-06-01`—`2029-12-31`，步长 86400 s）。见 `ephemeris-earth-202806-202912-*.json`。
- **小行星位置**：`mpc-2025WN6-response.json` 中 `Position.cartesianVelocity`（与上同一日心系）。
- **对齐方式**：`Position.epoch` 与各采样相对秒偏移统一换算到 UTC 绝对时刻后，对到达时刻 `ArrivalTime` 作**线性插值**，再计算日心系下 **地球位置矢量与小行星位置矢量之差的模**，记为 **`EarthAsteroidDistanceAtArrival_km`**（km）。

在撞击器可行解集合上，按 **该距离升序为主关键字**、**地球出发 Δv（LEO200）升序为次关键字** 排序，以体现「距离尽量小」；双器配对列表按 **撞击器到达距离升序**、再按 **观测器 dV_total 升序** 排列（`analyze_2025wn6.py`）。

## 6. 地球出发 Δv（任务书撞击器/观测器共用换算）

将 **`DV1_Mag` 视为双曲超速 v∞**，200 km 圆轨道高度对应近地点地心距 **rₚ = R⊕ + 200 km**：

**vₚ = √(v∞² + 2μ⊕/rₚ)**，**地球出发 Δv（LEO200）= vₚ − 10.96 km/s**。

其中 **μ⊕ = 3.986004418×10¹⁴ m³/s²**（实现见 `analyze_2025wn6.py`）。

观测器总指标：**dV_total = 地球出发 Δv（LEO200）+ DV2_Mag**（单位均为 m/s）。

## 8. 撞击器约束（光照、到达脉冲、**地球出发 Δv ≤ 1.1 km/s**、到达地–星距离）

撞击器须同时满足：

1. 到达光照角 **\< 80°**（第 4 节定义的角度 θ）  
2. **`DV2_Mag` \> 5000 m/s**（\> 5 km/s）  
3. **地球出发 Δv（LEO200，见第 6 节）≤ 1100 m/s**（即任务书 **dV 不超过 1.1 km/s**；由 `DV1_Mag` 换算，与观测器 `dV_total` 中的地球项定义相同）  
4. 在满足 (1)—(3) 的集合中，按 **到达时刻地–星距离尽量小**（第 5 节）排序；距离相同时 **地球出发 Δv** 更小者优先

| 阶段 | 条数 | 说明 |

## 12. 说明与局限

1. **5 天粗网格**：最优窗可能被细化步长（例如 1 天）局部刷新；需对 `/celestial/transfer` 重发请求。
2. **光照角**为 `RV2` 位置与 `DeltaV2` 矢量夹角（见第 4 节）；若需太阳照明几何，应追加日心方向矢量。
3. **Lambert 二脉冲模型**未包含行星引力阱细节，地球出发 Δv（LEO200）为任务给定近似。
4. **地–星距离**由日心位置样条按日步长线性插值得到；若需更高时间分辨率，可缩短 `Step` 或改用解析历元外推并与 API 对齐验证。  
5. **撞击器 1.1 km/s 上限**施加在 **地球出发 Δv（LEO200）** 上；若需更密时间格点，可能得到不同的可行域与配对数量。
