## 2. 到达月球轨道距离时的速度

飞行器沿双曲线向地球飞来，当再次经过地心距 r = 384,400 km（月球轨道距离）时，由能量守恒：

$$v = \sqrt{v_\infty^2 + \frac{2\mu_E}{r}} = \sqrt{0.6^2 + \frac{2 \times 398600.4}{384400}} \approx \mathbf{1.560\ km/s\ (1560\ m/s)}$$

**该速度与双曲线近地点瞄准高度无关**——同一 C3 下双曲线能量固定，在任意给定地心距处速度唯一确定。

### 2.1 近地点瞄准高度与近地点速度

近地点高度仅决定飞行器通过近地点时的速度，不改变在 r = 384,400 km 处的速度：


| 近地点瞄准高度 (km) | 近地点半径 r_p (km) | 近地点速度 v_p (m/s) | r = 384400 km 处速度 (m/s) |
| ------------ | -------------- | --------------- | ----------------------- |
| 1,000        | 7,378          | 10,412          | **1,560**               |
| 5,000        | 11,378         | 8,392           | **1,560**               |
| 10,000       | 16,378         | 7,003           | **1,560**               |
| 20,000       | 26,378         | 5,530           | **1,560**               |
| 30,000       | 36,378         | 4,720           | **1,560**               |


> 注：若采用月球辅助俘获，飞行器通常**在到达地球近地点之前**于月球附近完成飞越，实际不会到达上述双曲线近地点。

## 3. 月球飞越力学

### 3.1 飞越转向角

在月心双曲线飞越中，转向角 δ 与月心相对超速 v∞_M、近月点半径 r_p 的关系：

$$\delta = 2 \arcsin\left(\frac{1}{1 + r_p \cdot v_{\infty,M}^2 / \mu_M}\right)$$

其中 r_p = R_M + h_flyby（h_flyby 为距月面飞越高度）。

月心系速度增量模长：

$$|\Delta V|*{M} = 2  v*{\infty,M} \sin(\delta/2)$$

### 3.2 v∞ = 600 m/s 时飞越高度与转向角

（假设月心相对 v∞ ≈ 600 m/s，与地心 v∞ 接近的最优几何）


| 飞越高度 h_flyby (km) | 转向角 δ (°) | 月心系 |ΔV| (m/s) |
| ----------------- | --------- | -------------- |
| 100               | 123.6     | 1,057          |
| 300               | 120.9     | 1,044          |
| 500               | 118.4     | 1,031          |
| 1,000             | 112.7     | 999            |
| 2,000             | 103.4     | 942            |
| 3,000             | 95.8      | 890            |
| 5,000             | 84.0      | 803            |
| 10,000            | 65.0      | 645            |


600 m/s 的地心 v∞ 可被单次月球后方飞越基本抵消，实现俘获；飞越高度越低，转向角越大，俘获能力越强，但安全裕度越小。

## 8. 结论

1. **月球轨道距离处速度恒为 1560 m/s**，与双曲线近地点瞄准高度（1000~30000 km）无关。
2. **600 m/s 地心 v∞ 适合月球引力辅助俘获**，推荐飞越高度 **200~3000 km**。
3. **典型俘获结果**：近地点 1,000~30,000 km，远地点 ~378,000 km 的大椭圆。
4. **推荐方案**：200 km 飞越高度 → 近地点 ~3,300 km，远地点 ~378,000 km，e ≈ 0.951。
5. 若需近地点严格 ~~1,000 km 且远地点在月球轨道，推荐 **~~300 km** 飞越高度（俘获后近地点 951 km，远地点 379,918 km）。
